# 🌱 AgroTec Sustentável

Projeto desenvolvido para o Concurso Agrinho 2026.

## 📖 Sobre o Projeto

O AgroTec Sustentável tem como objetivo demonstrar como a tecnologia pode contribuir para uma produção rural mais eficiente e sustentável.

O projeto apresenta:

- Irrigação inteligente
- Uso de energia solar
- Automação rural
- Comedouro automatizado
- Simulador de economia de ração

## 🎯 Objetivos

- Reduzir desperdícios
- Melhorar a produtividade
- Incentivar a sustentabilidade
- Demonstrar soluções acessíveis para pequenos produtores

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
