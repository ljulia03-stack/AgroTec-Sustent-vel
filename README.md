# 🌱 AgroTec Sustentável 

Projeto desenvolvido para o Concurso Agrinho 2026 com o tema:

**“Agro forte, futuro sustentável”**

---

## 🚜 Sobre o Projeto

O **AgroTec Sustentável** é um projeto educacional que demonstra como a tecnologia pode ser aplicada no campo para promover sustentabilidade, automação e eficiência na produção rural.

Ele simula uma propriedade rural inteligente utilizando:

- 🌞 Energia solar  
- 💧 Irrigação automatizada  
- 🐄 Alimentação inteligente de animais  
- 📊 Controle de desperdício  
- 🌱 Práticas sustentáveis  

---

## 🎯 Objetivo

Mostrar como a tecnologia pode transformar a agricultura, trazendo benefícios como:

- Redução de desperdícios  
- Aumento da produtividade  
- Melhor uso dos recursos naturais  
- Menor impacto ambiental  

---

## 🧠 Funcionalidades do Projeto

✔ Site responsivo e moderno (dark mode)  
✔ Simulador de economia de ração  
✔ Estimativa de impacto ambiental (CO₂)  
✔ Gráfico de impacto ambiental  
✔ Galeria de imagens do projeto  
✔ Vídeo de apresentação  
✔ Animações suaves  

---

## 📊 Simulador Inteligente

O sistema calcula:

- Economia anual de ração  
- Redução estimada de CO₂  

### Fórmula utilizada:

```text
economia = ração diária × 365 × 0.3
