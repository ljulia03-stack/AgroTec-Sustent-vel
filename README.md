# 🌱 AgroTec Sustentável - Agrinho 2026

Projeto desenvolvido para o Concurso Agrinho 2026 com o tema:
**“Agro forte, futuro sustentável”**

---

## 🚜 Sobre o Projeto

O **AgroTec Sustentável** é um site educacional que demonstra como a tecnologia pode ser aplicada no campo para melhorar a produção agrícola e pecuária de forma sustentável.

O projeto simula uma propriedade rural inteligente utilizando:

- 🌞 Energia solar
- 💧 Automação de irrigação
- 🐄 Comedouro automatizado
- 📊 Simulador de economia de ração
- 🌱 Práticas sustentáveis

---

## 🎯 Objetivo

Promover a conscientização sobre o uso de tecnologia no campo, reduzindo desperdícios, aumentando a produtividade e respeitando o meio ambiente.

---

## 🧠 Funcionalidades do Site

✔ Página inicial com apresentação do projeto  
✔ Seção sobre agricultura inteligente  
✔ Demonstração de comedouro automatizado  
✔ Simulador de economia de ração  
✔ Layout moderno com tema escuro  
✔ Imagens reais de agricultura (online)  

---

## 🖥️ Tecnologias Utilizadas

- HTML5
- CSS3 (tema escuro moderno)
- JavaScript (interatividade)
- GitHub Pages (hospedagem)

---

## 📊 Simulador de Economia

O simulador calcula a economia anual de ração com base na redução de desperdício.

**Fórmula usada:**
economia = ração diária × 365 × 0.3

➡ Representa uma economia estimada de 30% com automação.
